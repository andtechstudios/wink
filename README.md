# Wink
***Win**dows Media **K**ey wrapper*

[![badge](https://img.shields.io/github/v/tag/andtechstudios/wink?label=nuget)](https://gitlab.com/andtech/pkg/-/packages?search[]=wink)

## Usage

### Configuration
Create a configuration file at `~/.config/wink.toml`.

```
default = "playpause"

[alias]
p = "playpause"
s = "stop"
b = "previous"
n = "next"
m = "mute"
```
